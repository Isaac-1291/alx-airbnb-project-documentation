# User Stories for Airbnb Clone Backend

Here are key user stories derived from the system's use case diagram:

1. **As a user**, I want to be able to register an account so that I can list or book properties.

2. **As a host**, I want to be able to list a new property with full details so that guests can book it.

3. **As a guest**, I want to search for properties based on location and price so that I can find a suitable place.

4. **As a guest**, I want to book a property for specific dates so that I can confirm my accommodation.

5. **As a guest**, I want to make secure online payments for my bookings so that my reservation is completed successfully.

6. **As a user**, I want to send and receive messages from other users so that I can communicate directly with hosts or guests.

7. **As a user**, I want to leave reviews after a stay so that other users can benefit from my experience.
